<p>Páginas disponiveis:</p>

<p>1-Página Home (home do cliente)</p>
<p>Essa página dá acesso a outras três páginas que o cliente pode acessar</p>
<img src="https://user-images.githubusercontent.com/100448388/234410833-40438b06-b6d8-4418-bd8e-262296a4db78.png">

<p>2-Página de cadastro do cliente</p>
<p>O cliente pode se cadastrar para utilizar a plataforma, os dados do cliente são salvos em arquivo JSON atraves do metodo POST</p>
<div align="center">
  <img src="https://user-images.githubusercontent.com/100448388/234412071-234963c7-512a-443c-aea1-2a3329d97e2f.png" width="80%">
</div>

<p>3-Página de Login</p>
<p>Para acessar o seu perfil é necessário que o cliente realize logim com os dados de seu cadastra.</p>
<p>Em caso de esquecimento de seu ID o cliente poderá clicar na opção "Esqueci meu ID" e recupera-lo ao informar seu CPF e senha. Já em caso de esquecimento da senha o cliente poderá clicar em "Esqueci minha senha", ao informar o CPF será enviado um email para seu email de cadastro com a respectiva senha</p>
<div align="center">
  <img src="https://user-images.githubusercontent.com/100448388/234415788-f08a23fe-2e5c-4fb5-9017-9836e4ed78c8.png" width="33%">

</div>

<p>4-Página de Perfil</p>
<p>Na página de Perfil o cliente pode ter acesso aos seus dados (utilização do metodo GET), atualizar suas informações (utilização do metodo PUT), vizualizar os pedidos de compra já realizados e cancelar os pedidos que ainda não foram entregues (utilização do metodo DELETE)</p>
